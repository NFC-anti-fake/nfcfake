业务流程说明

接口调用说明 :

1.商家url：

 ${merchant_url}（商户url描述示例http://xxxx.com）

 

2.请求方式：

application/json post请求

 

3.加密方式：

 

 

     请求参数字符串链接排序，A-Z排序加卡密匙MD5加密

     备注：密匙放置在请求Request Header  signature（卡密匙）：a22641417fbd87b76b25bbcaabc82731
